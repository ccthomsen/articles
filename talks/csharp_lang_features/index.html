  • Null-conditional operaters provide a short circuit to assign non nullable declared objects.
    • 
							var first = person?.FirstName;
  • No NullReferenceException will be thrown!
  • The null-conditional operators also work on delegates invoke methode
    • 
							this.SomethingHappened?.Invoke(this.eventargs);
  • Null-coalescing assign a the left hand value if it is not null, else the right hand value will be assigned.
    • 
							 first = person?.FirstName ?? "unspecified";
  • String interpolation enable embedded expressions in a string
    • 
							public string GetGradePointPercentage() => $"Name: {LastName}, {FirstName}. G.P.A: {Grades.Average():F2}";

  • Enhancement of interpolated verbatim strings means that the order of $ and @ doesn't matter 
    • 
								$@"..."
								@$"..."
    are booth valid interpolated verbatim strings. Verbatim string: 
    
								string filename1 = @"c:\documents\files\u0066.txt";
